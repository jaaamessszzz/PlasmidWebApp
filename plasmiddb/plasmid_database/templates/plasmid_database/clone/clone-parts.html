Instructions.

Part entry format (tab-delimited, one new line per part): Description  Part Type  Sequence  Alias (optional)
Recognized parts: 1 2 3 3a 3b 4 4a 4b 5 6 7 8 8a 8b
Define parts spanning multiple part types with a dash e.g. 2-4
Sequences are DNA (ATCGatcg)
Standardize parts: prepend/append nucleotides to certain part types so they conform to YTK MoClo standards

Notices for beta-testing.

This tool currently does not make any modifications to input sequences and will not accept sequences that do not conform to YTK MoClo standards. Please use a codon optimization tool (available from IDT, Benchling) to remove any BsaI/BsmBI/NotI restriction sites from your parts before adding them to the database!
